#  Personal Developer Portfolio

A modern, responsive personal portfolio showcasing my journey in software engineering. It highlights my solid foundation in back-end logic, database management, and UI integration through real-world web applications, as well as my ongoing transition into mobile development.

## Tech Stack
* **Framework:** React.js
* **Build Tool:** Vite
* **Styling:** Tailwind CSS
* **Icons:** React Icons & FontAwesome
* **Animations:** AOS (Animate On Scroll)****

## Key Features
* **Modern UI/UX:** Clean, dark-themed aesthetic with glassmorphism effects and tailored grid layouts.
* **Fully Responsive:** Seamlessly adapts to mobile, tablet, and desktop viewing experiences.
* **Project Showcase:** Detailed breakdowns of both public repositories and secure private internal systems.
* **Interactive Elements:** Smooth scroll animations, hover interactions, and dynamic content rendering.
